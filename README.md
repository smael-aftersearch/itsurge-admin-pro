# ITSurge Admin Pro

**Live Demo:** https://smael-aftersearch.github.io/itsurge-admin-pro/index.html

**ITSurge Admin Pro** is a premium, responsive, bilingual admin dashboard template designed for modern web applications. It includes polished English LTR and Persian RTL layouts, production-style pages, reusable UI components, and interactive demo behaviors using plain HTML, CSS, and JavaScript.

[Persian README](./README-FARSI.md)

## Overview

ITSurge Admin Pro is a static admin dashboard template for SaaS dashboards, CRM panels, ERP panels, e-commerce back offices, reporting dashboards, and bilingual admin products.

## Features

- English LTR and Persian RTL pages
- Responsive sidebar and topbar
- Notifications and profile dropdowns
- Users, Roles, Products, Orders, Reports and Settings pages
- Calendar, To-Do and Chat pages
- Authentication pages for login, register, forgot password, reset password and 2FA
- Add, edit, view and delete modals
- Confirmation dialogs and toast notifications
- Component pages for buttons, forms, tables, modals, tabs, dropdowns, selects, datepicker, charts, typography and notifications

## Run locally

Open `index.html` directly or serve the folder:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```
