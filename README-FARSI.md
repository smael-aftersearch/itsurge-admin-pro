# ITSurge Admin Pro

**Live Demo:** https://smael-aftersearch.github.io/itsurge-admin-pro/index.html

Persian RTL documentation for ITSurge Admin Pro.

[English README](./README.md)

## Features

- Persian RTL and English LTR layouts
- Responsive sidebar and topbar
- Users, Roles, Products, Orders, Reports, Settings, Calendar, To-Do, and Chat pages
- Authentication pages for login, register, forgot password, reset password, and 2FA
- Add, edit, view, and delete modals
- Toast notifications and table search
- Reusable component library

## Run locally

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.
