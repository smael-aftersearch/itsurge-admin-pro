# Security Policy

ITSurge Admin Pro is a static UI template. It does not include a backend, authentication server or database.

## Reporting security issues

If you find a security concern in the template code, open a private report in your repository or contact the maintainer.

## Notes for production use

- Replace all demo data before production use.
- Validate all form input on the server side.
- Implement authentication and authorization in your real backend.
- Do not rely on the demo JavaScript for real permission enforcement.
