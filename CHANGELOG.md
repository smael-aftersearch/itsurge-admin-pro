# Changelog

## 1.0.0 - 2026-06-22

### Added

- Initial public-ready release of ITSurge Admin Pro.
- English LTR and Persian RTL layouts.
- Entity-specific CRUD demo modals for users, products, orders, reports and roles.
- Calendar, To‑Do and Chat pages.
- Authentication menu with multiple variants for login, register, forgot password and 2FA.
- Full component library pages.
- GitHub-ready documentation, MIT license, contribution guide and QA checklist.

### Fixed

- Avatar initials are now centered and readable.
- Add/Create buttons open their correct modals.
- 2FA layouts no longer overflow on split/cover templates.
- Sidebar submenu behavior is improved for Components and Authentication.
